# Extism 插件生态系统

## 项目概述

这是一个基于 Extism 的插件生态系统实现，支持多语言 PDK (插件开发工具包)，提供完整的插件注册表和包管理功能。

## 主要功能

- 多语言 PDK 支持 (TypeScript, Go, Python, Rust, C/C++)
- 基本插件注册表功能
- 插件签名与验证
- 完整包管理系统与依赖解析
- Mastra 工具集成
- 现代化 Web UI 界面
